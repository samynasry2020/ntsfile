# NTSFM - File Manager

تطبيق لإدارة الملفات باستخدام PHP و MySQL.

## المتطلبات

- PHP 7.4 أو أحدث
- MySQL 5.7 أو أحدث
- Apache 2.4 أو أحدث

## التثبيت

1. قم بنسخ المشروع:
```bash
git clone https://github.com/yourusername/ntsfm.git
```

2. قم بتثبيت التبعيات:
```bash
composer install
```

3. قم بإنشاء قاعدة البيانات:
```bash
php create_tables.php
```

4. قم بتحديث ملفات التكوين:
- `config.php`
- `db.php`

## الاستخدام

1. قم بتسجيل الدخول باستخدام بياناتك
2. قم بإنشاء مجلدات جديدة
3. قم برفع الملفات
4. قم بمشاركة الملفات مع المستخدمين الآخرين

## الميزات

- إدارة الملفات والمجلدات
- مشاركة الملفات
- إنشاء روابط مشتركة
- إدارة الصلاحيات
- معاينة الملفات
- ضغط الملفات
